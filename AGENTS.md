## Architecture

Single-entry Go app at `cmd/laju-go/main.go`.

```
routes/web.go â†’ app/handlers/ â†’ app/services/ â†’ app/queries/ â†’ SQLite
```

- **Handlers**: Parse requests, call services, return responses. **Setiap module/feature harus handler file terpisah.**
- **Services**: Business logic, auth flows, external APIs. Boleh panggil `s.querier.*` atau `s.cache.*`.
- **Queries**: `app/queries/` is **generated by sqlc** from `queries/*.sql`. Never edit manually.
- **Cache**: `app/cache/` â€” In-memory session cache (sync.RWMutex + map).
- **Frontend**: Svelte 5 + Inertia.js. Entry at `frontend/src/main.ts`. Built to `dist/`.

## Three-Tier Rule (đź”´ CRITICAL)

**Handler â†’ Service â†’ Query â†’ DB.** Tidak ada layer yang boleh lompat.

| Layer | Boleh | Tidak Boleh |
|-------|-------|-------------|
| **Handler** | Parse request, panggil **Service**, return response | Panggil Query, akses DB, business logic |
| **Service** | Business logic, panggil `s.querier.*` atau `s.cache.*` | `sql.Open`, `db.Exec`, raw SQL |
| **Cache** | In-memory session cache (sync.RWMutex + map), dipanggil via Service | Akses langsung dari Handler |
| **Queries** | **SATU-SATUNYA** yang execute SQL | â€” |

## Svelte 5 Rules

- âťŚ Jangan `$effect` untuk derived state â†’ ganti `$derived()`
- âťŚ Jangan `$effect` untuk init state dari props â†’ `$state(value ?? default)`
- âś… `$effect` hanya untuk side effects: `document.title`, `localStorage`
- âś… Internal link WAJIB `use:inertia` dari `@inertiajs/svelte`
- đź”´ **fetch() CSRF header**: tiap `fetch()` ke `/app/*` atau `/admin/*` WAJIB `X-XSRF-TOKEN` dari `getCSRFToken()` (`lib/utils/helpers.ts`). Inertia's `router.*` auto-handle ini.
- Form submission pake `router.post()`/`router.put()`, bukan `<form>` biasa
- File upload via `fetch() + FormData`, simpan URL hasil via `router.put()`
- OAuth links (`/auth/google`, `/auth/github`) pake `<a>` biasa tanpa `use:inertia`

## HTTP Conventions

- POST/PUT redirect: `c.Redirect(path, fiber.StatusSeeOther)` (303, bukan 302)
- PUT/PATCH: return JSON untuk `fetch()`, redirect 303 untuk `router.put()`
- đź”´ **Body parsing**: WAJIB `c.BodyParser()` bukan `c.FormValue()`. Inertia kirim JSON.
- `fiber.Map` untuk adhoc response. Typed structs untuk service boundaries.
- CSRF middleware di `/app/*` routes. Axios auto-send `X-XSRF-TOKEN` dari cookie.
