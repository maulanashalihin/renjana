# Design Standards

Setiap halaman (SSR `.templ` maupun Inertia Svelte) harus:

1. **Visual Hierarchy** — Focal point jelas, ukuran/weight/kontras mengarahkan mata
2. **Whitespace lega** — Padding 16-24px minimum, jarak antar section 32-48px
3. **Typography** — Heading bold/berat, body clean/sans-serif. Tracking longgar untuk headline
4. **Color** — 60-30-10 rule. Netral dominan, aksen minimal
5. **Micro-interactions** — Hover state, transisi smooth, shadow halus
6. **Mobile-first** — Test di viewport 375px, 768px, 1440px

Kalau ragu dengan visual, minta screenshot via agent_browser.
